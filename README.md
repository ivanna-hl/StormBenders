We understand how frustrating it can be to go to great lengths to plan an outing or event with friends or family, only to have to cancel or postpone it due to unfavorable weather conditions.
That’s why we set out to develop a mobile app that not only predicts the weather using both ground-based and satellite observation data but also helps users plan outdoor activities safely and intelligently.
Additionally, we incorporated features designed to promote social and environmental awareness while ensuring users have a great time.
We recognized the importance of addressing this issue because canceling social events often causes not just emotional distress but can also result in financial losses. 
Moreover, going ahead with plans despite adverse weather can pose health risks.
To tackle this challenge, we built an application that leverages meteorological APIs to deliver real-time weather information to users, complemented by a chatbot that offers a personalized user experience. 
Furthermore, we developed a weather prediction algorithm based on historical data to enhance forecast accuracy.
